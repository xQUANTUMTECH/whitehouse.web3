How to Use These Placeholders

This file contains visual placeholders for the images used in the whitehouse.web3 website. To use actual images in your site:

  1. Create/obtain the following images:
    • logo.png - A logo for the website
    • friedman.jpg - An image of Milton Friedman
    • tariffs.jpg - A visualization of tariffs or trade
    • web3.jpg - An image representing Web3 or blockchain
    • future.jpg - An image representing future vision
  2. Place these images in the /assets folder
  3. They will automatically be used by the website

Until then, viewing the website in a browser will show these placeholders instead of actual images.
